## Installation and Setup Instructions

#### Example:

Clone down this [repository](https://github.com/cleandevcode/git-explorer). You will need `node` and `npm` or `yarn` installed globally on your machine.

Installation:

`yarn`

To Start Server:

`yarn start`

To Visit App:

`http://localhost:3000`
